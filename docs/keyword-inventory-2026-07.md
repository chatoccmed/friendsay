## ลำดับขยาย (ข้อเสนอ)

คลื่น 1 (100 คีย์เวิร์ด): เครื่องใช้ไฟฟ้าภายในบ้าน + เครื่องใช้ในบ้าน (ต่อยอดฐานเดิม ค่าคอมสูงสุด)
คลื่น 2: ความงาม + สัตว์เลี้ยง (ค่าคอมสูง ซื้อซ้ำ)
คลื่น 3: อิเล็กทรอนิกส์ + เกม + กล้อง (สเปกเทียบง่าย)
คลื่น 4+: ตามข้อมูล GA จริง

## งานถัดไป

1. เก็บหมวดย่อย (L2/L3) ของหมวดคลื่น 1-2
2. เก็บคำค้นยอดนิยม (Shopee search suggestion + Google) รายหมวดย่อย
3. จัดเกรดแต่ละคีย์เวิร์ด: volume / ค่าคอม / ฤดูกาล / คู่แข่ง
4. ผูกกับสถานะ Open API (ดูหมายเหตุด้านล่าง)
